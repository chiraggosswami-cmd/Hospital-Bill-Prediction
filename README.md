# Hospital-Bill-Prediction
To predict the Hospital Bill Prices (regression) based on the various features present in the dataset, currently the pipeline only uses Machine Learning Models [Random Forest (R2 Score of 91) and Decision Trees (R2 Score of 82)]. 

The project uses open-source dataset. The link is provided here: 
url = "https://raw.githubusercontent.com/YBIFoundation/ProjectDataSet/main/Hospital%20Bill%20Prediction%20for%20MediCarePlus.csv"
